# 마개조 내용

## 단축키 추가함
- 상단 숫자에 맞춰서 방향키 입력되도록 수정 
  - 1 2 3 4 5 6 7 8 9 전부 만듦. 
- 키 2개 동시입력 전부 만듦
  - 기존 LPRPLKRK에 해당하는 QWAS의 위아래에 추가함. 
  - Z는 RP+LK
  - X는 LP+RK
  - E는 AP
  - F는 AK
- 이미지로 저장 단축키 추가. 상단 숫자 1 왼쪽 ` 버튼임
- R 은 스크류, F는 대시로 매칭 시킴

## 기타 이미지에 추가함. 
- 세비지, 이후 콤보 내용 추가함.
